# 系统架构设计文档

## 1. 概述

本文档描述 RAG 知识库系统的架构设计。

## 2. 核心组件

### 2.1 Ollama + bge-m3
- **用途**: Embedding 向量化
- **模型**: BAAI/bge-m3 (1024维)
- **部署**: 本地部署，MPS加速

### 2.2 Qdrant
- **用途**: 向量数据库
- **特点**: 高性能、Collection隔离
- **存储**: 本地磁盘

### 2.3 FastAPI
- **用途**: RESTful API
- **特性**: 异步、自动文档

## 3. 数据处理流程

```
文档上传 → 文本提取 → 智能分块 → 向量化 → 存储 → 检索
```

## 4. 项目隔离

每个项目对应一个 Qdrant Collection，实现数据完全隔离。
